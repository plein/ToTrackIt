Features

📈

Process Tracking

Easily register processes and set durations or deadlines to monitor progress in real-time.

🔔

Webhook Notifications

Get a webhook notification the moment a deadline is missed, with the process context you need to diagnose issues quickly.

📊

Metrics & Dashboards

Measure completion times and missed deadlines in the built-in UI, and scrape Prometheus metrics into Grafana.

🔍

Rich Context and Tags

Add descriptions, context, or customer-specific IDs to each process and assign tags for granular analysis.

🔌

API-First + Web UI

Register and complete processes through a REST API with OpenAPI docs, or manage everything from the clean built-in dashboard.

🐳

Self-Hosted & Open Source

Run the whole stack with Docker Compose — API, UI, PostgreSQL, and monitoring. Apache 2.0 licensed.
