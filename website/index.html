Know when async processes miss their SLO — and why

Account activations, payments, KYC checks — when they silently get stuck, customers notice first. ToTrackIt tracks every run against its deadline, feeds the alerting stack you already use, and shows which country, channel, or provider the problems concentrate in. Open source and self-hosted.

Features

📈

Track Every Run Against Its SLO

Two API calls per process — start and complete. ToTrackIt computes on-track, missed, on-time, and late status in real time.

📟

Alert With the Tools You Already Use

Deadline-aware Prometheus metrics plug into Datadog monitors, Prometheus alerts, and metric-based SLOs — no new pager, no migration.

🔍

Root Cause by Tags

When something breaks, see instantly that the stuck and slow runs share a segment — country, channel, provider — with impact and p90 latency per tag.

🔔

Alerts That Land on the Answer

Deadline-missed webhooks carry a deep link straight to the affected process page: all runs, impacted tags, and latency for the period you pick.

🔌

API-First + Web UI

Register and complete processes through a REST API with OpenAPI docs, or manage everything from the clean built-in dashboard.

🐳

Self-Hosted & Open Source

Run the whole stack with Docker Compose — API, UI, PostgreSQL, and monitoring. Apache 2.0 licensed.
